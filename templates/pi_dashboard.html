Methodologies

AlQahtani Method

Atlas-based approach using dental development stages for both primary and permanent dentition. Covers ages 0-23 years.

Demirjian Method

Scoring system based on 7 mandibular permanent teeth. Uses an 8-stage maturity scale (A-H) with weighted scoring.

Assessment Protocol

01 // REVIEW

Examine the orthopantomogram for dental development.

02 // ASSESS

Determine development stage for each tooth.

03 // CALCULATE

Apply methodology to estimate chronological age.

04 // SUBMIT

Record final age estimation for blinded case.

NOTE: You will not have access to patient identities or actual ages during the estimation process.
