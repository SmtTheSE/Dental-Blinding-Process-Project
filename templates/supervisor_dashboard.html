Blinding Workflow Status

The blinding workflow ensures scientific rigor by preventing the Principal Investigator from accessing actual patient information or knowing which method is being used for each case.

  • Patient data is stored in the Master Sheet (only accessible to supervisors)
  • Randomized codes are assigned to each patient for both methods
  • Data is shuffled and blinded before sharing with the PI
  • PI performs age estimation using only codes and OPG images
  • Supervisor unblinds data and performs final analysis
